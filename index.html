Projeto em Destaque

S.I.G.M.A

Full Stack

Sistema Integrado de Gerenciamento de Manutenção e Abastecimento

Sistema corporativo para gestão integrada de frota veicular, desenvolvido para otimizar o controle de abastecimento e manutenção de veículos da instituição.

Principais Funcionalidades

  • Gestão de Frota: Controle completo de veículos (VTR) com cadastro, edição e upload de fotos
  • Sistema de Empréstimo: Solicitação e aprovação de empréstimo de veículos entre unidades
  • Controle por UETC: Cada usuário visualiza apenas os dados de sua unidade específica
  • Níveis de Acesso: Sistema hierárquico com permissões diferenciadas por cargo
  • Manutenção Preventiva e Corretiva: Controle completo com status em tempo real
  • Planos Personalizados: Criação de pacotes de manutenção customizados por administradores
  • Análise de Eficiência: Cálculo automático de eficiência de combustível e custos de manutenção por família de veículos
  • Importação de Dados: Upload de arquivos com processamento automatizado via Python/Pandas
  • Relatórios e Dashboards: Visualização de dados com gráficos e relatórios personalizados

Stack Tecnológica

Node.js Express MongoDB JWT Python Pandas REST API
